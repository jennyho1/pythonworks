In programming you often need to know if an expression is True or False. For example, the statement "4 > 8" is false because 4 is not bigger than 8.

Symbol Description
< smaller than
<= smaller or equal to
> greater than
>= greater or equal to
== equals

Here are some examples to get you started:

>>> 10 == 10 True >>> 3 >= 4 False
